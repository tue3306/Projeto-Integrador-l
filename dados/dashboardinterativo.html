Alerta Urbano - Dashboard Interativo Aprimorado

Este dashboard interativo apresenta dados reais e atualizados sobre infraestrutura urbana no Distrito Federal, com base em fontes oficiais como IBGE, DETRAN-DF e GDF. Os dados são relevantes para o projeto "Alerta Urbano", que visa monitorar e reportar problemas como buracos, calçadas danificadas e falta de iluminação. Use os filtros para explorar tendências por ano e categoria. Explicações detalhadas acompanham cada seção para facilitar a compreensão e narração em vídeos.
